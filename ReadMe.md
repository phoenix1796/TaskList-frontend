# Simple Task List frontend with JWT Bearer Auth

## Installation

```
yarn install
```

## Steps to run

* Change API_URL in src/utils/Config.js as per need. (https://github.com/phoenix1796/SpringBoot-TaskList)

* For development purposes

```
yarn serve
```

or

* For deployment purposes

```
yarn build
```
